# 🍔 Madrugadão Lanches - Cardápio Digital

🚀 **Acesse o cardápio online:** [https://tinyurl.com/madruagadaolanches](https://tinyurl.com/madruagadaolanches)

## 📖 Sobre o Projeto
Este é um Cardápio Digital responsivo e moderno, desenvolvido com foco na experiência do usuário (UX) e inspirado nas interfaces limpas de grandes aplicativos de delivery (estilo iFood/cards). 

O sistema foi criado para automatizar e facilitar os pedidos de uma lancheria, permitindo que o cliente navegue pelos produtos, personalize seus lanches e envie o pedido formatado com o valor total diretamente para o WhatsApp do estabelecimento.

## ✨ Funcionalidades
*   **Interface Responsiva:** Design otimizado para funcionar perfeitamente em smartphones e computadores.
*   **Carrinho de Compras Inteligente:** Cálculo automático em tempo real do valor total do pedido.
*   **Personalização de Lanches (Modais):** Sistema avançado onde o cliente pode montar porções ou remover ingredientes indesejados do seu Xis, Cachorro-Quente ou Pastel (ex: "Sem Milho", "Sem Maionese").
*   **Montagem de Combos:** Lógica estruturada para escolha de múltiplos itens dentro de uma oferta (Combo da Galera).
*   **Checkout via WhatsApp:** Redirecionamento automático com uma mensagem contendo o resumo completo do pedido e o valor final a pagar.

## 🛠️ Tecnologias Utilizadas
*   **HTML5:** Estruturação semântica e acessível.
*   **CSS3:** Estilização baseada em Flexbox e CSS Grid para layouts dinâmicos, com paleta de cores voltada para o nicho de alimentação (vermelho/laranja).
*   **JavaScript (Vanilla):** Lógica de programação para manipulação do DOM, controle do array de produtos no carrinho, validação de itens e formatação da API do WhatsApp.
